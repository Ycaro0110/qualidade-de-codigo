# Exercício – Qualidade de Código

Ycaro entriel marques de oliveira

Qualidade de código refere-se ao conjunto de boas práticas que tornam um software:

✔ Mais legível

✔ Fácil de manter

✔ Seguro

✔ Testável

✔ Escalável

✔ Eficiente

Um código de qualidade não é apenas aquele que funciona, mas aquele que:

Pode ser entendido por outros desenvolvedores

Pode ser modificado sem quebrar outras partes

Segue padrões e boas práticas

Trata erros adequadamente

É preparado para crescer

linguagem escolhida : JAVA
